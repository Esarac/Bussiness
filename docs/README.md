Un accionista desea creemos un programa que logre modelar en su totalidad un holding empresarial. Para esto la aplicación debe: Agregar cualquier tipo de empresa al holding empresarial, Anadir productos a las empresas de producción, calcular el impuesto procultura de las empresas educativas, calcular cuántos arboles deben sembrar las compañías de medicamentos y mostrar toda la información de la empresa de holding (Incluyendo toda la información de sus clases subordinadas). Todo esto debe estar incluido en el programa junto con el API y toda su documentación.
